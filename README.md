# Spring Boot Example With Containerization

## Getting Started

Clone the repo down and run locally.

You can hit the [endpoint](localhost:8080/v2/api-docs) to view all the APIs exposed.

This [endpoint](http://localhost:8080/swagger-ui.html) will make it look much nicer.
